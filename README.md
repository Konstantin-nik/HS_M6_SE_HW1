## Task 1
Given an integer x, return true if x is a palindrome, and false otherwise. 

### Additional information
Input and Output: CLI
Input: only numbers < 10^8

### Example
**Input**: x = 121 \
**Output**: true

**Input**: x = 10 \
**Output**: false

### Time
Expected time: 10 min \
Real Time: 5 min
